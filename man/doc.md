# NAME
zet - Terminal Zettelkasten Manager
# SYNOPSIS
**zet**
[**new** *"note name"* *--path*]
[**daily** *--fzf-options*]
[**new-entry**]
[**tomorrow**]
[**yesterday**]
[**tags** *--fzf-options*]
[**search** *--fzf-options* *--folder*]
[**delete** *--fzf-options**]
[**grep**]
[**help**]
# DESCRIPTION
zet is a cli zettelkasten tool for Obsidian.

You can find the source code at https://github.com/skykosiner/zet.
